## Step use Web Ban9Jin
## Harry Potter
### 1. Clone
##### ทำการ Clone Code ต่างๆมาที่เครื่องคอม
```
$ git clone https://github.com/Champoolwoo/midtest.git
```
### 2. Install
##### ทำการติดตั้ง Packet ที่ต้องใช้
```
$ npm install
```
### 3. Run
##### ทำการ run
```
$ npm start
```
### 4. Test
##### ทำการ run test
```
$ npm test
```
